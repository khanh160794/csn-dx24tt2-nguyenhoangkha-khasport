1. [cite_start]**Môi trường:** Cài đặt phần mềm **LocalWP** để tạo máy chủ ảo cục bộ[cite: 117].
2. [cite_start]**Cài đặt:** Khởi tạo một trang web WordPress mới thông qua giao diện LocalWP[cite: 117].
3. [cite_start]**Mã nguồn:** Sao chép các tệp tin trong thư mục `/src` của Repository này vào thư mục dự án trên máy[cite: 83].
4. [cite_start]**Cấu hình:** Cài đặt và kích hoạt Plugin WooCommerce, sau đó cấu hình đơn vị tiền tệ (VNĐ) và khu vực vận chuyển[cite: 115, 118].
5. [cite_start]**Dữ liệu:** Import dữ liệu mẫu từ thư mục `/setup` để hiển thị danh mục và sản phẩm đúng như báo cáo[cite: 81].

## 6. Tổ chức cây thư mục Repository
Tuân thủ quy định của học phần, Repository được tổ chức như sau:
* [cite_start]**`/src`**: Chứa các tập tin nguồn và tập tin dữ liệu thử nghiệm[cite: 83].
* [cite_start]**`/progress-report`**: Chứa các báo cáo tiến độ thực hiện hàng tuần (Bắt buộc)[cite: 84].
* [cite_start]**`/thesis`**: Chứa các tập tin tài liệu văn bản của Đồ án (thư mục con `/doc` và `/pdf`)[cite: 85, 87, 88].
* [cite_start]**`/setup`**: Chứa các tập tin dùng để cài đặt chương trình và dữ liệu mẫu[cite: 81].
* [cite_start]**`/refs`**: Chứa các tài liệu, văn bản dùng để tham khảo[cite: 91].
