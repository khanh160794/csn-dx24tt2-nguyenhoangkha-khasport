## 5. Hướng dẫn cài đặt và Triển khai (Setup)
1. **Môi trường:** Cài đặt phần mềm **LocalWP** để tạo máy chủ ảo cục bộ.
2. **Cài đặt:** Khởi tạo một trang web WordPress mới thông qua giao diện LocalWP.
3. **Mã nguồn:** Sao chép các tệp tin trong thư mục `/src` của Repository này vào thư mục dự án trên máy.
4. **Cấu hình:** Cài đặt và kích hoạt Plugin WooCommerce, sau đó cấu hình đơn vị tiền tệ (VNĐ) và khu vực vận chuyển.
5. **Dữ liệu:** Import dữ liệu mẫu từ thư mục `/setup` để hiển thị danh mục và sản phẩm đúng như báo cáo.

## 6. Tổ chức cây thư mục Repository
Tuân thủ quy định của học phần, Repository được tổ chức như sau:
* **`/src`**: Chứa các tập tin nguồn và tập tin dữ liệu thử nghiệm.
* **`/progress-report`**: Chứa các báo cáo tiến độ thực hiện hàng tuần.
* **`/thesis`**: Chứa các tập tin tài liệu văn bản của Đồ án (thư mục con `/doc` và `/pdf`).
* **`/setup`**: Chứa các tập tin dùng để cài đặt chương trình và dữ liệu mẫu.
* **`/refs`**: Chứa các tài liệu, văn bản dùng để tham khảo.
